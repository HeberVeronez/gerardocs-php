# Sistema de Exportação de Dados

Projeto backend desenvolvido em PHP para cadastro de dados em banco de dados e exportação das informações em formato XML ou TXT, simulando integração com sistemas externos.

## Funcionalidades
- Cadastro de registros
- Armazenamento em banco de dados relacional
- Exportação dos dados em arquivos XML ou TXT
- Interface simples para interação com o sistema

## Tecnologias Utilizadas
- PHP
- MySQL
- JavaScript
- HTML
- Bootstrap

## Objetivo do Projeto
Projeto criado com foco no aprendizado de desenvolvimento backend, manipulação de dados, integração entre frontend e backend e geração de arquivos para troca de informações entre sistemas.

## Como Executar o Projeto
1. Clonar o repositório
2. Configurar o banco de dados
3. Executar o projeto em um servidor local (ex: XAMPP ou WAMP)

## Observações
Projeto com fins educacionais, desenvolvido para prática de conceitos fundamentais de backend.
